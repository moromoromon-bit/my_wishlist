# 🛍️ my_wishlist

사고 싶은 물건을 모아두고 관리하는 개인용 위시리스트 웹앱입니다.
모든 데이터는 브라우저(localStorage)에만 저장되며 서버로 전송되지 않습니다.

## ✨ 기능

- 🔗 **링크에서 자동 불러오기(개선)** — 다중 프록시 + 메타태그/JSON-LD 기반으로 대표 이미지·제목 추출 시도
- 📌 **북마클릿 모달 안내** — 상단 `📌 북마클릿` 버튼으로 드래그 등록 링크 + 수동 복사용 코드 제공
- 🖼️ **카드형 목록** — 이미지·이름을 사각형 카드로 나열, 이미지를 클릭하면 상품 링크가 새 탭에서 열림
- ✏️ **수정 / 🗑️ 삭제**
- 📝 **메모** — 왜 사고 싶은지 기록
- 🛒 **구매함 탭 이동** — 사면 구매함으로 이동
- ⭐ **후기 + 별점** — 구매 후 후기와 1~5점 별점 기록
- 💰 **가격 입력 + 총합** — 위시/구매 금액 합계 표시
- 🔥 **우선순위(불꽃 3단계)** — 🔥 끌림 / 🔥🔥 사고 싶음 / 🔥🔥🔥 꼭 사고싶음
- 🏷️ **카테고리/태그** + 필터
- 🔍 **검색 / 정렬** — 이름·메모·태그 검색, 최신·가격·우선순위·이름순 정렬
- 🌙 **다크모드**
- 📅 **날짜 기록** — 추가일·구매일
- 📤 **백업 / 📥 복원** — JSON 파일로 내보내기·불러오기

## 🚀 사용 방법

### 1) 그냥 로컬에서 쓰기
`index.html` 파일을 다운로드해서 더블클릭으로 열면 됩니다.

### 2) GitHub Pages로 호스팅하기 (추천)
1. 저장소 **Settings → Pages** 이동
2. **Source**: `Deploy from a branch`
3. **Branch**: `main` / `/ (root)` 선택 후 **Save**
4. 잠시 후 `https://moromoromon-bit.github.io/my_wishlist/` 에서 접속 가능

### 3) 북마클릿 설치/사용 (권장)
1. 상단 액션의 **📌 북마클릿** 버튼을 누릅니다.
2. 모달의 **`📌 위시리스트에 담기` 링크를 즐겨찾기 바로 드래그**해 등록합니다.
3. 드래그가 어려우면 모달의 **📋 코드 복사**로 `javascript:...` 코드를 복사해 수동으로 즐겨찾기 URL에 붙여넣습니다.
4. 임의의 상품 페이지에서 해당 즐겨찾기를 클릭하면 위시리스트가 열리며 자동 추가됩니다.

> 네이버 스마트스토어, 쿠팡 등 일부 사이트는 CORS/차단 정책 때문에 자동 불러오기가 실패할 수 있습니다.  
> 이 경우 북마클릿 사용을 권장합니다.
